- mymenu -


SUMMARY:

This files add top navigation bar into admin section of your module.


BACKGROUND:

Some newer modules has navigation above its admin's section.
But once someone click "preferences", the navigation will be lost.

mymenu integrates your modules's preferences seamlessly into your 
module's admin section.


HOW TO USE:

1) copy mymenu.php to modules/(your module)/admin/

2) edit all of your admin's files and add include('./mymenu.php'); 
   after xoops_cp_header();

3) edit admin/index.php and insert branch code for altsys after including cp_header.php


This is a sample skelton of admin's file.

[code]
require_once( '../../../include/cp_header.php' ) ;

// branch for altsys
if( defined( 'XOOPS_TRUST_PATH' ) && ! empty( $_GET['lib'] ) ) {
	$mydirname = basename( dirname( dirname( __FILE__ ) ) ) ;
	$mydirpath = dirname( dirname( __FILE__ ) ) ;

	// common libs (eg. altsys)
	$lib = preg_replace( '/[^a-zA-Z0-9_-]/' , '' , $_GET['lib'] ) ;
	$page = preg_replace( '/[^a-zA-Z0-9_-]/' , '' , @$_GET['page'] ) ;
	
	if( file_exists( XOOPS_TRUST_PATH.'/libs/'.$lib.'/'.$page.'.php' ) ) {
		include XOOPS_TRUST_PATH.'/libs/'.$lib.'/'.$page.'.php' ;
	} else if( file_exists( XOOPS_TRUST_PATH.'/libs/'.$lib.'/index.php' ) ) {
		include XOOPS_TRUST_PATH.'/libs/'.$lib.'/index.php' ;
	} else {
		die( 'wrong request' ) ;
	}
	exit ;
}


(DB update section)

xoops_cp_header() ;
include(dirname(__FILE__).'/mymenu.php');

(output section)

xoops_cp_footer() ;
[/code]

That's all !

●要旨

モジュールの管理画面すべてに、トップメニューをつけるためのファイル群です。


●背景

最近のモジュールには、管理画面用メニューを搭載したものが多く見られますが、そのいずれも、一般設定に入ると、途端にメニューが消えてしまい、しかも、一般設定を更新した後は変なページに飛ばされてしまいます。

mymenuは、一般設定画面にもメニューを表示し、設定変更後も、その設定画面に戻れるようにするために作りました。

0.18からは、altsysがインストールされている場合には、altsys内のblocksadminやtplsadminへのリンクも表示します。


●使い方：

1) mymenu.php を modules/(dirname)/admin/ にコピーします。

2) 管理画面用のファイルを一通り編集し、include('./mymenu.php'); という行を、xoops_cp_header(); の直後に挿入します。

3) admin/index.php のcp_header.php読込直後に、altsysへの分岐処理を挿入します。


管理画面用スクリプトのサンプルスケルトン：
[code]
require_once( '../../../include/cp_header.php' ) ;

// branch for altsys
if( defined( 'XOOPS_TRUST_PATH' ) && ! empty( $_GET['lib'] ) ) {
